# Terraform-Assignments

This repository contails terraform module for following AWS services:
 - AWS SageMaker
 - AWS BedRock
 - AWS Athena
 - AWS Glue
 - AWS DataZone
 - AWS RedShift

# Prerequistites
 - AWS Account
 - VPC
 - Subnets
 - Security Group
 - IAM User with all required permissions
